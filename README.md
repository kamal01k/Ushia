# Ushia

Project for [Synthia-dataset][synthia].

This project uses [OpenStreetMap][OpenStreetMap] OSM data.

## Install

	git clone https://github.com/marcgpuig/Ushia.git

## Unity version
Currently using [Unity3D][unity] 5.5.1.
