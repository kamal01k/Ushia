# Ushia

On development project for [Synthia-dataset][synthia].

This project uses [OpenStreetMap][OpenStreetMap] ([OSM XML][OSMXML]) and [Mapzen][Mapzen] data.

![Cerdanyola]

	git clone https://github.com/marcgpuig/Ushia.git
